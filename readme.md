
# 🚀 Self-Healing AIOps System (ML + LangGraph)

An intelligent system that predicts machine failures using Machine Learning and autonomously takes recovery actions using a LangGraph-based agent.

---

## 🔥 Features

- 📊 Failure prediction using ML (RandomForest)
- ⚙️ Real-time monitoring simulation
- 🧠 LangGraph-based decision agent
- 🚨 Multi-level risk detection (low / warning / critical)
- 🔧 Automated actions (restart, alert, logging)
